Software I use:

  • I use GNU/Linux as my primary Operating System, specifically Arch Linux distribution.
  • Neovim as my text editor with lua plugins to make it a full fledged and superior IDE.
  • window manager: dwm It's so good! I love my workflow using it. I maintain my own build of dwm and was one of the first projects I started working on after my Arch install because it added to the 'building it from ground up' view. I'm currently quite satisfied with my build, though there can always be improvements.
  • terminal: st the simple terminal or the suckless terminal is a pretty great terminal! with quite alot of features packed in just 2656 SLOC! Much less than XTerm with 65K SLOC most being legacy code.
    However, without patches it is quite unusable for me. So, with a few patches in my build of st I try to make it more featureful and it does the job incredibly well!
    It comes with multiple fonts support,ligature support,Pywal support and has external-pipe patch through which I can use certain scripts to say
    • copy the output of a command to system clipboard or
    • open links from my terminal-output to my browser or
    • copy the link to clipboard and selecting them using dmenu
  • video player: mpv minimal yet most powerful media player I've ever seen! love using my keybindings.
  • image viewer: sxiv Handles all the images very well, it is very featureful and hackable with it's scripting capability using unix piping and key-handler to run custom commands or other scripts
  • browser: Brave or Ungoogled Chromium and Librewolf or GNU Icecat
  • piped:a pretty good youtube alternative. On my phone I use LibreTube , a FOSS android application which serves as a lightweight front-end for piped. I might self-host this in future using Invidious project. I personally recommend my normie friends who wish to use just plain youtube but hate the youtube android app to use NewPipe! //I dislike youtube vanced as it still associates w the original app.
