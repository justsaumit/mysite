Hardware/OS/WM I use:

Software I use:

  • I use GNU/Linux as my primary Operating System, specifically Arch Linux distribution.
  • Neovim as my text editor with lua plugins to make it a full fledged and superior IDE.
  • window manager: dwm It's so good! I love my workflow using it. I maintain my own build of dwm and was one of the first projects I started working on after my Arch install because it added to the 'building it from ground up' view. I'm currently quite satisfied with my build, though there can always be improvements.
  • terminal: st with quite a few patches and pywal support in my st, it does the job.
  • video player: mpv minimal media player with my personal keybindings.
  • image viewer: sxiv Handles all the images very well, it is very featureful and hackable with it's scripting capability using unix piping and key-handler to run custom commands or other scripts
  • browser: Brave or Ungoogled Chromium and LibreWolf
  • piped: a pretty good youtube alternative. On my phone I use LibreTube , a FOSS android application which serves as a lightweight front-end for piped. I might self-host this in future using Invidious project.
